Dao

下面以User表为例进行说明

1、新增

User user = new User();

UserMapper mapper = MapperFactory.getMapper(UserMapper.class);

int n = mapper.insert(user);

 

2、删除

1、删除一条记录

UserMapper mapper = MapperFactory.getMapper(UserMapper.class);

int n = mapper.deleteByPrimaryKey(userPojo.getId());

 

2、删除多条记录

UserroleMapper userroleMapper = MapperFactory.getMapper(UserroleMapper.class);

 

UserroleExample example = new UserroleExample();

example.createCriteria().andUseridEqualTo(userPojo.getId());

userroleMapper.deleteByExample(example);

 

3、修改

1、全值修改（完全根据java bean对象来修改表记录，如果bean的属性为null，则会把表记录相应字段置为null）

UserMapper mapper = MapperFactory.getMapper(UserMapper.class);

 

int n = mapper.updateByPrimaryKey(user);

 

2、激活值修改（只把java bean中不为null的字段更新到表记录中）

UserMapper mapper = MapperFactory.getMapper(UserMapper.class);

 

int n = mapper.updateByPrimaryKeySelective(user);

 

注意：关注红色字体部分，其方法名是不一样的。

6、戒律

1、所有dao操作必须在service目录中

2、需要事务控制的方法名必须参考db.xml中的配置。

3、因为我们的DAO层是代码生成器生成的，为了减少以后表结构变更带来的程序修改，请大家遵循下面这条规则：

1、不要在XxxMapper.java中添加任何东西

2、在XxxMapper.xml中添加自定义的sql时，请添加在最后，而且要和生成的空上5-10行。

3、可以看一下com.pica.wificity.util.ServiceUtils

 

Service

1、Service类都必须位于com.pica.wificity.service包中
2、命名规则：XxxxService，其中的Xxxx根据业务规则来划分，各开发人员自己掌握，一般根据表名或模块名来命名。
3、典型的Service类的写法(注意类的声明处有注解)

@Component

@Transactional

public class UserService

{

    private static Logger log = LoggerFactory.getLogger(UserService.class);

   

    /**

     *

     * @param userPojo

     * @return

     */

    public Long add(User user)

    {

        UserMapper mapper = MapperFactory.getMapper(UserMapper.class);

        int n = mapper.insert(user);

        return n;

    }

 

    /**

     * 删除记录

     * @param userPojo

     * @return

     */

    public int delete(User user)

    {

        UserMapper mapper = MapperFactory.getMapper(UserMapper.class);

        int n = mapper.deleteByPrimaryKey(user.getId());

        UserroleMapper userroleMapper = MapperFactory.getMapper(UserroleMapper.class);

 

        UserroleExample example = new UserroleExample();

        example.createCriteria().andUseridEqualTo(user.getId());

        userroleMapper.deleteByExample(example);

 

        return n;

}

 

/**

 * 修改记录

 */

public int updateByPrimaryKey(User user)

    {

        UserMapper mapper = MapperFactory.getMapper(UserMapper.class);

        int n = mapper.updateByPrimaryKey(user);

        return n;

 }

}
